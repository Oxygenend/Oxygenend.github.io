术语 解释说明
database 数据库，可建立多个
collection 集合，一组数据，相当于js里的数组
document 文档，一条具体的数据，相当于js里的对象
filed 字段，文档中的属性名称，相当与js中的对象属性

如何连mongoDB接数据库

  1. 下载mongoos第三方模块npm install mongoose 启动 ner start mongoDB
  2. 引入mongoose第三方模块，用来操作数据库
  3. 用connect方法来连接数据库，参数为mongodb://localhost/playground，最后一个是数据库名写上就会自动创建
  4. 分别用.then和.catch对数据库连接成功或失败一个反馈
const mongoose = require('mongoose')
mongoose.connect('mongoose://localhost/playground')
    .then(() => console.log('数据库连接成功'))
    .catch(() => console.log('数据库连接失败'))

如何创建一个集合以及向集合插入文档

  1. 创建集合规则，用new实例化mongoose，用Schema方法添加规则，规则为字段名和类型
  2. 创建集合，model方法第一个参数是集合名第二个参数是集合规则
  3. 创建文档，用new实例化集合，参数为一个对象每一项是具体的字段和值
  4. 将文档保存到数据库，文档名.save()
const personSchema = new mongoose.Schema(&#123;
    name: String,
    age: Number,
    hobbies: [String]
&#125;)
const Person = mongoose.model('Person', personSchema)
const person = new Person(&#123;
    name: 'zhangsan',
    age: 12,
    hobbies: ['sing', 'read']
&#125;)
person.save()

find()查询指定集合的所有文档

Person.find().then(result => console.log(result))

find({条件})查询指定集合中满足条件的文档

Person.find(&#123;name:'李四'&#125;).then(result=>console.log(result))
find(&#123;字段:&#123;$gt:开始范围 , $lt: 结束范围&#125;&#125;)`查询字段位于之间的文档 `&lt小于` `&gt大于

Person.find(&#123;age:&#123;$gt:20,$lt40&#125;&#125;).then(result=>console.log(result))
find(&#123;字段:&#123;$in:包含项&#125;&#125;)

Person.find(&#123;hobbies:&#123;$in:['sing']&#125;&#125;).then(result=>console.log(result))

find().select('字段1，字段2..')查询指定字段，字段前加-，可以去除字段

Person.find().select('name,age,-id').then(result=>console.log(result))

find().sort('排序字段')默认按升序排序，加-按降序排序

集合关联

通常不同集合的数据之间是有关系的，例如文章信息和用户信息存储在不同集合中，但文章是某个用户发表的，要查询文章的所有信息包括发表用户，就需要用到集合关联。

  • 使用id对集合进行关联
  • 使用populate方法进行关联集合查询
const userSchema = new mongoose.Schema(&#123;
    name: &#123;
        type: String,
        required: true
    &#125;
&#125;);
const postSchema = new mongoose.Schema(&#123;
    title: String,
    author: &#123;
        //类型为id，固定写法
        type: mongoose.Schema.Types.ObjectId,
        //关联集合user
        ref: 'User'
    &#125;
&#125;)
const Post = mongoose.model('Post', postSchema);
const User = mongoose.model('User', userSchema)
Post.create(&#123; title: '123', author: '5c0caae2c4e4081c28439791' &#125;).then(result => console.log(result));
User.create(&#123; name: '李四' &#125;).then(result => console.log(result))
    //populate()关联集合查询
Post.find().populate('author').then(result => console.log(result))
